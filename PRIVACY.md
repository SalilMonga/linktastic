# Privacy Policy for Linktastic

**Effective date: March 24, 2026**

Linktastic is a Chrome extension that copies the current page's title and URL as a rich HTML hyperlink to your clipboard. This policy explains what data the extension accesses, what it does with it, and what permissions it needs.

## What data we collect

None. Linktastic does not collect, store, or transmit any data. There are no analytics, no tracking, no cookies, and no user accounts. Everything happens entirely on your device.

## How we use permissions

Linktastic requests the following Chrome permissions:

- **activeTab** -- Lets the extension read the title and URL of the tab you are currently viewing, but only when you click the extension icon or use the keyboard shortcut.
- **scripting** -- Lets the extension run a small script on the active tab to write the rich link to your clipboard.
- **clipboardWrite** -- Lets the extension place the formatted hyperlink onto your clipboard.
- **host_permissions (`<all_urls>`)** -- Lets the extension run on any page so it can copy a link regardless of which site you are on.

These permissions are used solely to perform the copy action you initiate. No data from any page is read in the background or sent anywhere.

## Third-party services

Linktastic does not use any third-party services, APIs, or servers. It has no network activity at all.

## Changes to this policy

If this policy changes, the updated version will be published in the project repository. The effective date at the top of the page will be updated accordingly.
